## Haversine Formula

$$distance_{c_1, c_2}=2r\cdot sin^{-1}\left(\sqrt{hav(lt_2-lt_1)+cos(lt_1)\cdot cos(lt_2)\cdot hav(lg_2-lg_1)}\right)$$
$$hav(\theta)=sin^2\left(\frac{\theta}{2}\right)$$
